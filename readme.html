Acerca del Proyecto

Descripción del Proyecto

Este proyecto tiene como propósito integrar herramientas tecnológicas en el proceso educativo, fortaleciendo la planeación, implementación y reflexión docente con apoyo de las TIC. Busca generar experiencias educativas significativas, mejorar la calidad del aprendizaje y fomentar la creatividad y autonomía en estudiantes de distintos niveles escolares.

Objetivos

  • Promover el uso pedagógico de las TIC en el aula.
  • Facilitar la documentación y reflexión sobre la práctica docente.
  • Desarrollar materiales y recursos educativos digitales innovadores.

Metodología

La intervención se organiza en cinco ejes: Planeación, Implementación, Documentación de materiales o recursos, Evaluación y Reflexión. Cada eje permite registrar información, añadir recursos digitales, insertar código externo, imágenes y archivos descargables, asegurando un seguimiento completo del proceso educativo.

Alcance

El proyecto abarca desde sexto de primaria hasta tercero de secundaria, permitiendo la adaptación de estrategias según el nivel educativo y promoviendo la integración progresiva de herramientas TIC en el aprendizaje.
